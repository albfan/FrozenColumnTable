# Frozen Column Table

## Description
A table that shows several columns as an always visible row header

## Screenshots

![screenshot](etc/frozenColumnTable.png)

## Execution

Example provides a main class for jar and mvn exec so you can run either with:

```
mvn exec:java
```

or

```
mvn package
java -jar target/*.jar
```
